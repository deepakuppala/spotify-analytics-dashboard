# 🎧 Spotify Analytics Dashboard

🚀 **An interactive data science dashboard built using Streamlit to analyze Spotify music data and generate smart recommendations.**

---

## 📸 Dashboard Preview

![Dashboard Screenshot](screenshot.png)

---

## ✨ Features

* 📊 **KPI Metrics** – Total songs, artists, genres, popularity
* 🎯 **Sidebar Filters** – Artist, genre, popularity
* 📈 **Interactive Visualizations** – Built with Plotly
* 🤖 **Smart Recommendation System** – Based on audio features
* 🎧 **Clean Professional UI** – Industry-style dashboard

---

## 🧠 Machine Learning

* Used **cosine similarity** for recommendations
* Optimized memory usage using **on-demand computation**
* Features used:

  * Danceability
  * Energy
  * Tempo
  * Loudness

---

## 🛠️ Tech Stack

* 🐍 Python
* 📊 Pandas
* 🌐 Streamlit
* 📈 Plotly
* 🤖 Scikit-learn
